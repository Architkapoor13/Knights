# Knights
The famous Knights and Knaves puzzle implemented in Python!

### Description
It includes a total of 4 puzzles, in each of these puzzles there are some people which state something and based on their statements the program tels you if that peron is a Knave or a knight.
**Knave** = A person who always speaks lie.
**Knight** = A person who always speaks only the truth.

### How to run
```Bash
python puzzle.py
```
